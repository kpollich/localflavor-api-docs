# Images

## Fields 

| Name | Type | Description |
|------|------|-------------|
| cdn_url | string | URL source for the image |


## List Deals

```http
GET /images HTTP/1.1
Accept: application/vnd.api+json
```

## Retrieve a Deal by ID

```http
GET /images/:image_id HTTP/1.1
Accept: application/vnd.api+json
```
